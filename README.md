# Raffaele Scapati Portfolio

Questo è il sito web personale di Raffaele Scapati, uno sviluppatore web italiano. Il sito presenta il portfolio, il curriculum vitae e un modulo di contatto.

## Descrizione

Il progetto è un sito web statico costruito con HTML, SCSS e Bootstrap. Include sezioni per presentare l'autore, i suoi progetti, passioni e hobby, oltre a una pagina dedicata al CV e al contatto.

## Caratteristiche

- Design responsive
- Utilizzo di SCSS per gli stili
- Integrazione con Bootstrap per componenti UI
- Favicon personalizzate
- Form di contatto

## Struttura del Progetto

```
raffaelescapati97.github.io/
├── index.html          # Pagina principale
├── cv.html             # Curriculum Vitae
├── contattami.html     # Pagina di contatto
├── assets/             # Risorse statiche
│   ├── favicon_io/     # Favicon e manifest
│   └── ...             # Immagini
├── src/
│   └── scss/           # File SCSS
│       ├── main.scss   # File principale SCSS
│       └── ...         # Altri file SCSS
├── dist/
│   └── css/            # CSS compilato
└── README.md           # Questo file
```

## Come utilizzare

1. Clona il repository:
   ```
   git clone https://github.com/raffaelescapati97/raffaelescapati97.github.io.git
   ```

2. Installa Dart SASS (scarica da https://github.com/sass/dart-sass/releases e estrai in una cartella accessibile).

3. Compila SCSS in CSS:
   ```
   path\to\sass.bat src/scss/main.scss dist/css/main.css
   ```

4. Apri `index.html` nel browser.

## Build

Il progetto utilizza SASS per compilare SCSS. Per compilare:

```
sass src/scss/main.scss dist/css/main.css
```
